# 特征提取维度定义

本文件定义从优质对话片段中提取前任特征的维度和方法。

## 说话风格层（Style Features）

### 幽默感
- **识别方式**：「哈哈」「笑死」「绷不住」「离谱」「抽象」「hhh」「😂」「🤣」等信号词频率
- **分级**：
  - high：每3条消息出现1次以上幽默信号
  - medium：每10条出现1次左右
  - low：几乎不出现

### 话题引导深度
- **识别方式**：开放式提问频率「你觉得」「你怎么看」「如果是你」「为什么」
- **分级**：high / medium / low（同上）

### 共情表达
从对方消息中提取以下类型的表达：
- 反射式回应：「原来如此」「听起来你...」「感觉你...」
- 情感确认：「能理解」「我懂」「我知道你...」
- 安慰行为：「抱抱」「没事的」「辛苦了」

### 关心细节
- 主动询问状态：「你今天」「最近怎么样」「吃了吗」「睡了吗」
- 记住小事并提起：「上次你说」「之前你提到」「你跟我说过」

### 倾听信号
「继续说」「然后呢」「真的假的」「详细说」「后来呢」「说来听听」

## 性格维度层（Personality Features）评分标准（1-10分）

| 维度 | 评分依据 |
|------|---------|
| 主动性（initiative）| 对话片段中对方主动发起的比例 |
| 情绪稳定性（stability）| 优质对话占总对话的比例（越高越稳定）|
| 共情能力（empathy）| 共情表达信号密度 |
| 幽默感（humor）| 幽默信号出现频率 |
| 关心细节（care）| 关心类信号出现频率 |
| 耐心（patience）| 倾听信号密度 |
| 话题深度（depth）| 开放式提问频率 |

## 标签翻译表

用户可能用以下口语标签描述前任，AI 应将其翻译为具体行为特征：

| 用户标签 | 行为特征 |
|---------|---------|
| 话痨 | 消息密度高，经常连发多条，话题跳跃快，不等对方回就继续说 |
| 幽默 | 善用自嘲和情境幽默，能用玩笑化解尴尬 |
| 贴心 | 记住对方说过的小事，主动跟进，细节关心 |
| 秒回 | 立刻回复，期待对方也秒回 |
| 高冷 | 回复简短，不主动发起话题，但被逗开心时会暴露 |
| 话少但走心 | 回复不频繁，但内容有质量、有深度 |
| 理性 | 分析型回应，善于帮对方梳理问题 |
| 感性 | 情绪化表达，容易被触动，共情能力强 |
| 逗比 | 频繁使用网络梗，喜欢搞笑，气氛活跃 |
| 深情 | 表达细腻，重视关系，话语中有很多情感成分 |

## 化学反应评分（chemistry_score）

综合加权计算：

```
chemistry = (
    initiative × 0.15 +
    stability  × 0.15 +
    empathy    × 0.20 +
    humor      × 0.15 +
    care       × 0.15 +
    patience   × 0.10 +
    depth      × 0.10
)
```

分值范围：1-10，分越高代表对话化学反应越好。
